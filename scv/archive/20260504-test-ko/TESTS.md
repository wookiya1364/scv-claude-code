# 테스트 플랜 — 결제 페이지에 환불 버튼 추가

## 개요

환불 흐름의 E2E + 단위 테스트.

## 테스트 시나리오

### T1. 7 일 이내 환불 성공

- **준비**: 3 일 전 결제된 주문, Stripe 결제 완료
- **실행**: 환불 버튼 클릭
- **기대**: 환불 처리됨, 이메일 발송
- **통과 기준**: HTTP 200, refund 이벤트 발생

### T2. 7 일 초과 환불 거부

- **준비**: 10 일 전 결제된 주문
- **실행**: 환불 버튼 클릭
- **기대**: HTTP 403, 에러 메시지 표시
- **통과 기준**: 환불 처리 안 됨

## 실행 방법

```bash
npm run test:e2e -- refund
```

## 통과 기준

- 모든 시나리오 통과
- 커버리지 ≥ 80%
